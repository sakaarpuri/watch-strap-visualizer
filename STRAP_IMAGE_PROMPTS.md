# Strap Image Prompt Library (Two-Part Assets)

Use these prompts in your image-generation tool (e.g., Nano Banana) to create strap assets for the Watch Strap Visualizer.

## Global Requirements (prepend to every prompt)
- Output TWO separate PNG files with transparent background: `part-a` (12 o'clock side with buckle/hardware if applicable) and `part-b` (6 o'clock tail side).
- Product-only image, no watch case, no dial, no hand, no wrist, no person, no props, no text, no logo, no watermark.
- Flat front view, perfectly centered, vertical orientation, realistic texture, soft studio lighting, crisp edges.
- Keep identical camera angle, framing, scale, and lighting across all color variants of the same style.
- High resolution: minimum 2000px height per part.

---

## Strict Negative Prompt (append to every generation)
Do not generate any watch dial, watch case, crown, wrist, human, hand, skin, sleeve, table, background scene, text, logo, watermark, shadows outside product, perspective tilt, or diagonal angle.

---

## Output Naming Convention
For each style + color/finish variant, export:
- `style-variant-part-a.png`
- `style-variant-part-b.png`

Examples:
- `leather-smooth-black-part-a.png`
- `leather-smooth-black-part-b.png`
- `steel-brushed-part-a.png`
- `steel-brushed-part-b.png`
